Frequently Asked

Which versions of Windows are supported?

Folio works on Windows 10 and Windows 11 (64-bit). No additional dependencies are required.

Does it support images?

Yes! This is Folio's superpower. It can link images from a local folder to your document templates automatically.

Is there a cloud version?

No. We intentionally built Folio as an offline tool to ensure maximum data privacy for sensitive university records.
