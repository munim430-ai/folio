How it works

Three steps to done

No training required. If you can fill an Excel sheet, you can use Folio.

STEP 01
⬇️

Download the Excel template

Select your university, grab the pre-formatted Excel file with every required column already labelled.

STEP 02
📝

Fill in your student data

Paste student records into the sheet — name, passport, sponsor details, photo path. One row per student.

STEP 03
⚡

Click Generate

Drop the file into Folio, hit Generate. Every student gets their own perfectly filled DOCX file instantly.

Features

Everything you need

Built specifically for education consultants handling Korean university applications.

📄

Bulk DOCX Generation

Generate one properly-formatted DOCX per student from a single Excel upload. Works for any batch size.

📸

Automatic Photo Embedding

Specify a photo path in Excel and Folio embeds the student's photo directly into the correct form cell.

✅

Pre-filled Checkboxes

Semester (Summer), consent forms, and agreement checkboxes are ticked automatically in every document.

🌐

100% Offline

No internet required after installation. Your student data never leaves your machine.

🛡️

Validation Built-in

Folio flags rows with missing required fields before generating — no silent blank forms.

🎓

Mokwon University Ready

Full support for Mokwon University's 3-page admission packet — admission form, consent, and affidavit.
