Installation

Folio is a portable Windows app.

  1. Download the app
  2. Move to a folder
  3. Run Folio.exe
Download Folio
Windows may show SmartScreen → click "Run anyway"

Quick Start

  1. Download template
  2. Fill Excel
  3. Upload file
  4. Click Generate

Excel Setup
